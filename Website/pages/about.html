About My Site

The aim of FitForce is to spread information to help fitness enthusiasts better manage their lifestyle. For this purpose, nutritional and workout plans, tips and tools, are added, to help their fitness journey.

FitForce has been developed solely using HTML, CSS & JavaScript. There are a couple of things which this project taught me:

  • Prior to this project, my experience with website design included designing landing pages. This project challenged me to develop a website with multiple pages coherently using a variety of website design elements such as forms, tables, feeds, social media button, and the trending dark mode.
  • This project required me to use CSS and HTML rather than other advanced frameworks. While it is time consuming to do everything from scratch, it is amazing how powerful these tools are and allow us to have control over every element in our website.
  • What I learnt the most through this project is the aspect of design itself. It is easier for me to implement the code for the website being a coder. However, designing is not my forte and this project helped me explore my designing skills.

I would have preffered loading HTML scripts using javascript, as it would aid to avoid rewriting a lot of code for the header and footer elements. Furthermore, I would spend more time making my design more clean and user friendly.
